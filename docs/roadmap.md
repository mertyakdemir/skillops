# SkillOps Roadmap

## MVP

- Discover common AI agent instruction files in repositories.
- Report stale references to files, directories, commands, and docs.
- Detect duplicated instruction blocks across files.
- Identify conflicting guidance between repository-level and nested
  instructions.
- Provide a local-first CLI workflow with useful human-readable output.

## Later

- Machine-readable report formats.
- Configurable rule severity.
- Ignore patterns and scoped scans.
- Package publishing and release automation.
- Editor and CI integrations after the CLI and core scanner are stable.
