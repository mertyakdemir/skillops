# Changelog

All notable changes to SkillOps will be documented in this file.

## 0.1.0 - 2026-05-26

### Added

- Initial local-first CLI scanner for AI agent instruction files.
- Discovery for `AGENTS.md`, `CLAUDE.md`, Codex, Cursor, Copilot, and `docs/ai` instruction files.
- Checks for missing owner metadata, stale review metadata, broken file references, duplicate instructions, and package manager command conflicts.
- Human-readable scan output plus JSON stdout and JSON file reports.
- Sample repository fixtures and CI documentation for running the workspace CLI.
